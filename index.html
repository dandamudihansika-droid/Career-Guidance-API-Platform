Project Presentation Portal

CareerGuide API

A modular full-stack career advisor and resume skill extractor powered by weighted matching algorithms, SQLite storage, and secure RESTful endpoints.

Backend REST API Endpoints

GET /api/recommendations
POST /api/profile
PUT /api/profile/skills
DELETE /api/profile/resume

How to Run Locally

# 1. Activate Virtual Environment
.\venv\Scripts\Activate.ps1
# 2. Seed SQLite Database (8,483 records)
python init_db.py
# 3. Start Backend Server
python app.py
Launch Local Application (http://127.0.0.1:5000)
